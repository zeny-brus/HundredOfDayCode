Atributos globais

class

As classes permitem que CSS e JavaScript selecionem e acessem elementos específicos por meio de seletores de classe. 

        
            <p class="note editorial">
            Above point sounds a bit obvious. Remove/rewrite?
            </p>

contenteditable

é um atributo enumerado que indica se o elemento deve ser editável pelo usuário. Nesse caso, o navegador modifica seu widget para permitir a edição.

        
            <p contenteditable="true">Este conteudo é editavel</p>

Este conteudo é editavel

data-*

atributo de dados personalizados permitem que as informações proprietárias sejam trocadas entre o HTML e sua representacao DOM por scripts 

        
            <li data-id="10784">Jason Walters, 003: Found dead in "A View to a Kill".</li>

Hidden

As entradas ocultas são completamente invisíveis na página renderizada e não há como torná-las visíveis no conteúdo da página. 

        
            <input type="hidden" id="postId" name="postId" value="34657" />

id

um identificador que deve ser unico em todo o documento. Sua finalidade é identificar o elemento ao vincular script ou stilo 

        
            <p id="postId">exemplo de codigo com id </p>

style

O elemento HTML contém informações de estilo para um documento ou parte de um documento.

        
            <p style="color: blue">The style attribute can override it, though.</p>

The style attribute can override it, though.

tabindex

Utilizado para focalizar elementos HTML, geralmente com valor de uma chave, que determina a ordem relativa da navegação, podendo ser usado a tecla TAB 

        
            <div tabindex="0">ordenação </div>
            <div tabindex="1">ordenação 1 </div>

title

O elemento HTML define o título do documento que é mostrado na barra de título de um navegador ou na guia de uma página. Ele contém apenas texto 

        
            <title>titulo da pagina</title>
